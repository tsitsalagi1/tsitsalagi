Verified author request

This does not instantly delete a post. It verifies the private author email and code, then sends the request to the admin sheet for review.

Why a code is required

Only the original author should be able to request removal, mark a listing sold, or mark an issue resolved. The private code prevents someone else from removing another person’s post just because they can see it online.

  • Your private email is not shown publicly.
  • Your removal/update code is not shown publicly.
  • Requests are reviewed before public changes are made.

Lost your code? Use the Contact / Report page and include the public title, your private author email, and what needs to change.
